== Description ==

An example of rendering an additional link after the WooCommerce Add to Cart button based on custom field data set for a product (external_url, external_url_label).

The article [Using Custom Fields with WooCommerce Products](https://www.itthinx.com/2023/01/05/using-custom-fields-with-woocommerce-products/) provides a tutorial on how to use this child theme.
